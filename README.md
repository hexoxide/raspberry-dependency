# raspberry-dependency
Automatic dependency installation and management for the raspberry pi 3 B+

The following tools will be installed by using this tool.

| Library/Tool | Version    |
|--------------|------------|
| FairMQ       | 1.1.5      |
| ZeroMQ       | 4.2.1      |
| Zookeeper    | 3.4.9      |
| Cmake        | 3.11.0     |
| Boost        | 1.66.0     |
| Yaml-cpp     | 0.5.2      |
| Compiler     | gcc 6.3.0  |
